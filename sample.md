# Dracula Theme for MacDown

A dark theme for rendered markdown using the official [Dracula](https://draculatheme.com) color palette.

### Table

| Color  | Hex       | Usage    |
|--------|-----------|----------|
| Purple | `#BD93F9` | Headings |
| Cyan   | `#8BE9FD` | Links    |
